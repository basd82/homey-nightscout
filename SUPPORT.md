# Support

For support, bug reports and feature requests, use GitHub Issues:

https://github.com/basd82/homey-nightscout/issues

Please include:

- Homey model
- App version
- Nightscout version if known
- Whether you use mmol/L or mg/dL
- A screenshot of the error if available
- Relevant Homey app log output, without secrets or tokens

Never share your Nightscout API secret or token publicly.
